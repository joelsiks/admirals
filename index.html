Welcome to Admirals!

A game engine for developing performant 2D real-time strategy (RTS) multiplayer games.

The Admirals engine is a specialized platform crafted for the creation of high-performance 2D real-time strategy (RTS) multiplayer games. Unlike conventional engines that cater to a broad spectrum of game types, Admirals places a deliberate focus on key elements crucial for RTS game development.

Our engine is designed to streamline the creation process, emphasizing the most essential features specifically tailored for RTS games. By prioritizing performance, intuitive design, and robust multiplayer capabilities, Admirals offers developers a more efficient and user-friendly experience.

Goals: (1) Develop a game engine, (2) Create a small game using the engine.
